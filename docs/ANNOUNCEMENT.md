# 魔法師西蒙 繁體中文化 — 介紹文案

> 一份可直接用於發布貼文 / 社群 / 論壇的介紹文案,說明本專案的貢獻。

---

## 短版(一段話)

《魔法師西蒙》(Simon the Sorcerer,1993)——那個滿嘴刻薄話、活像少年版黑爵士的小巫師,終於有了**完整繁體中文版**。
我們沒有把它做成無聲的字幕版,也沒有像前人那樣對著殘缺的資料硬翻:我們把 **1993 floppy 版的完整文字**
和 **CD talkie 版的英語原聲**融合在一起,做出一個原版從來不存在的組合——**完整中文字幕 + 完整英語配音,還能按 F8 中英即時切換**。
當年你看不懂的那些英式冷笑話,現在一句不漏。

---

## 我們做了什麼(貢獻)

**1. 解決了一個沒人診斷出來的根本問題:英文 CD 版根本沒有完整字幕。**

當年 Adventure Soft 把純文字的 floppy 版加上語音做成 CD 版時,為了塞進 170MB 配音,
把大量對白的文字**從資料裡拿掉了**。結果你若在 CD 版打開字幕,會看到角色**嘴巴在動、卻一個字都沒有**。
這不是 bug,是資料先天就缺了約四成。前一版中文化對著這個殘缺來源翻,再怎麼努力都補不回那四成。

**2. 用「融合」補齊:floppy 完整文字 × CD 英語語音。**

- 字幕來源改用**文字完整的 floppy 版**(每句都在),
- 語音沿用 **CD 版的英語配音**(Chris Barrie 的原聲),
- 兩者用引擎腳本的行序對齊接起來——用 ScummVM 自己的反組譯器把兩版腳本 dump 出來,逐句配對。

**3. 全劇 4035 條文字,一句不漏地翻成繁體中文。**

對白、物件名、系統訊息、操作選單——連開場那道「唸咒防拷」都自動填答免查手冊。
中文用 24×24 點陣正常字級,不是糊成一團的小字。遊戲中按 **F8** 就能中英字幕即時切換,語音維持英語。

**4. 全程 patch ScummVM 引擎,不改動任何遊戲原始檔。**

## 那些你當年沒看懂的笑話,回來了

魔法師西蒙的靈魂全在對白裡——它對經典奇幻的惡搞,尤其是對**托爾金**的致敬與吐槽,是整部遊戲最好笑的部分:

- 在森林深處的洞穴裡,一群 cosplay 成哈比人的「托爾金鑑賞協會」正在開年會,一個曬到中暑、扮成咕嚕的傢伙釣著魚。你若敢說一句「托爾金是誰?」,換來的是暴怒:

> **「托爾金是有史以來最偉大的作家!你總聽過《手指飾品之王》吧?那《毛腳矮人》呢?」**
> *(Lord of the Finger Ornaments = 魔戒;The Short Man with Hairy Feet = 哈比人——連書名都在惡搞。)*

- 而當你惹毛了森林裡的女巫,她拔劍相向,喊出的是這句經典到不行的台詞:

> **「以托爾金的神聖鬍鬚之名,準備受死吧!!!」**
> *(By the sacred beard of J.R.R. Tolkien, prepare to DIE!!!)*

還有跟你大談「這是歧視——第三級的!」的會說話蠹蟲、按劇本演出卻搞起勞資糾紛的巨魔、
花三十枚金幣入會只拿到一個「巫師小子錢包」的巫師公會⋯⋯這些英式冷笑話的節奏與無厘頭,都盡量照原味譯出。

當年我們一手滑鼠、一手翻著三大誌零星的圖文攻略,笑一半、猜一半。現在,整場戲都補完了。

## 跟前一版有什麼不同

| | 前一版 | 本版 |
|---|---|---|
| 翻譯條數 | 1237(分母還是捏造的) | **4035(全劇完整)** |
| 字幕來源 | 英文 CD 版(缺四成) | **floppy 完整版** |
| 語音 | 無 | **CD 英語原聲** |
| 中/英切換 | 無 | **F8 即時切換** |
| 中文字級 | 12×12(糊,還是假的 16×16) | **24×24 真點陣** |
| 操作選單 | 英文(漏做) | **繁體中文** |
| 防拷 | 要查手冊 | **自動 bypass** |

前一版的失敗不在中文品質,而在**動手前沒把中文化的範圍調查清楚**。詳見 [失敗分析](PREVIOUS_REPO_POSTMORTEM.md)。
